# World Cup Corner Radar

Initial import of the World Cup Corner Radar project.

This repository contains:
- FastAPI backend for World Cup corner analytics
- Docker-based frontend and backend setup
- Analyst Mode with deterministic analysis and Ollama-backed chat
- SQL foundation files for v2 roster/data-quality work
- Utility scripts and project docs

Notes:
- Sensitive runtime secrets are excluded.
- Use `.env.example` as the template for local or VPS setup.
- Database dumps and local backups are intentionally not committed.
